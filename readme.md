# DIY Grooming Blog – Server-Backend

T3A2-B MERN Full-stack App

Setup:

`node version v16.13.0 (lts version)`

To update:
- `nvm install --lts`
- `nvm use --lts`

`npm version 8.1.0`

Once cloned, run:
```
npm install
```
